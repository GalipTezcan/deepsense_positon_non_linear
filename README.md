---

## Folder Structure

The **Scenario 32** folder consists of three sub-folders:

### 1. `unit1`
Includes the data captured by unit 1:
- `camera_data`
- `mmWave_data`
- `GPS_data`
- `LiDAR_data`
- `radar_data`

### 2. `unit2`
Includes:
- `GPS_data` captured by unit 2

### 3. `resources`
Includes:
- Scenario-specific additional information

---

## Additional File

The Scenario 32 folder also includes:

- `scenario 32.csv`

This CSV file contains the paths to all the collected data.  
For each coherent time, the corresponding:
- Visual data
- Wireless data
- GPS data

are provided.
